Content

eds-sidebar is optimized for the following content elements:

  • eds-menu-items
  • eds-sidebar-groups
  • h3, h4, h5 tags
  • hrs

When dealing with collapsable sidebars, you can provide an icon for non eds-menu-item elements by nesting content inside an eds-sidebar-group and setting an icon for the group. This will cause the entire group to collapse down to a single icon.
