eds-button components are applied to native buttons as attributes (<button eds-button>) and limited to styles only.

Configuration Options Sets the inverse style for the given visual motif of the button.
Sets the visual motif of the button. This is generally used to convey intent. See related design principles for more information.

Available Values:

  • basic
  • primary
  • tertiary
